# **App Name**: AI Canvas

## Core Features:

- Hero Section: Full-screen cinematic introduction with name and role display: Aatif Muneeb Khan, AI / ML Engineer. Focus: Research-driven AI systems, model evaluation, machine learning, deep learning, and production ML pipelines.
- About Section: Scroll-triggered animation with detailed information about skills and experience.
- Projects Section: Horizontal scrolling project showcase with animated transitions between projects.
- Research & Writing Section: Display research papers and articles with an option to read or download.
- Contact Section: Minimal contact form and details for easy communication. Use Firestore to capture all submitted inquiries.
- Dynamic Project Loading: Load and manage project content dynamically from Firebase Firestore.
- Generative project suggestion: An AI powered 'tool' to generate an innovative project based on current ML trends for presentation in the project section. This provides inspiration.

## Style Guidelines:

- Background color: Dark charcoal (#222222) for a cinematic and elite aesthetic.
- Primary color: Soft grey (#DDDDDD) for text, ensuring readability against the dark background.
- Accent color: Pale blue (#99BADD) to highlight key elements and interactive components.
- Font pairing: 'Space Grotesk' for headings, a techy sans-serif, and 'Inter' for body text, a modern and readable sans-serif.
- Framer Motion animations for smooth fade-ins, zoom effects, and transitions.
- Parallax effect on hero section text and background elements, linked to mouse movement.
- Full-screen sections with smooth scroll snapping for a clean and immersive user experience.
